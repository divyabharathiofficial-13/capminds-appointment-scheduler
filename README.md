# CAPMINDS Appointment Scheduler

## Overview

A responsive Appointment Scheduling application developed using pure HTML, CSS, and JavaScript as part of the CAPMINDS Frontend Assessment.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage

## Features

- Responsive UI (Mobile, Tablet, Desktop)
- Monthly Appointment Calendar
- Book Appointment
- Edit Appointment
- Delete Appointment
- Search by Patient Name
- Search by Doctor Name
- Filter by Date / Date Range
- Appointment Dashboard
- Local Storage Persistence
- Form Validation

## Project Structure

```
├── index.html
├── Styles.css
├── script.js
└── img/
```

## How to Run

1. Download the project.
2. Open `index.html` in any modern browser.

No installation or backend required.
